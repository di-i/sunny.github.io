# Thousand sunny

Test website. Adaptive landing page of travel site.

## Objective 

1. Learn how to work with frameworks and libraries:
   - Bootstrap 4.5
   - Jquery
   - Animate CSS
   - Wow JS
   
2. Implement a fixed burger menu





## Stack 

- HTML5
- CSS3
- JS
- Methodology BEM
- Git
